                  *********************************Relational Database Management System*****************************************

-->Every web application contains three parts.
   1)Front end application(UI)
   2)Backend application(Business logic)
   3)Database(store)

-->Databases are maily two types:
   1)Relational DB's:Oracel MSSQL P.SQL
   2)No-sql DB's: MongoDB cachebase Hbase Dynamo DB

Maily we can work on this 5db's
1)Aurora
2)RDS
3)Dynamo
4)Elastic cache
5)Document DB

mainly RDS is divided into 3v's
1)Volume = db's IN TB
2)Variety = Relational/No-sql
3)Velocity = Speed 


RDS used to represent the structural data like data having clear boundries
   ID   Name    Email
    1   Vinay  xyz@gamil.com

In RDS we will store the data in the form of tables.
Table contains Rows and columns.

Example:Student data
 
     ID     Name     Email
     1      Vinay     xyz
     2      sai       xyz

Procedure for create DB's:
1)Goto RDS
2)Choose create DBS option
3)select standard creation
4)select engine type as MSSQL
5)Free tair
6)identifier name
7)public access as yes
8)Addition config-->Enter DB name


Note:
once DB launched it will generate end point for out DB to connect.
